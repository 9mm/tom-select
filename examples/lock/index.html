Locking Example

Controls can be locked to prevent user interaction.

  • JavaScript
  • HTML 
(tomSelect('#select-locked-single')).lock();
<select id="select-locked-single" placeholder="No input allowed...">
	<option value="A">Option A</option>
	<option value="B" selected>Option B</option>
	<option value="C">Option C</option>
</select>
  • JavaScript
  • HTML 
(tomSelect('#select-locked')).lock();
<select id="select-locked" multiple placeholder="No input allowed...">
	<option value="A">Option A</option>
	<option value="B" selected>Option B</option>
	<option value="C" selected>Option C</option>
</select>
